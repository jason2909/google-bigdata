BigQuery es la base de datos de análisis de bajo costo, NoOps y totalmente administrada de Google. Con BigQuery, puede consultar terabytes y terabytes de datos sin tener que administrar ninguna infraestructura ni necesitar un administrador de base de datos. BigQuery usa SQL y puede aprovechar el modelo de pago por uso. BigQuery le permite concentrarse en analizar datos para encontrar información valiosa.
